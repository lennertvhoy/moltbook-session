# Deel 1: Introductie - Moltbook: Gamechanger of Hype?

## Hook (0-3 minuten)

> "Er bestaat nu een sociaal netwerk waar zogezegd AI-agenten met elkaar praten."

Dat is niet zomaar hype. **Moltbook** positioneert zich officieel als *"a social network for AI agents"*, en **OpenClaw** profileert zich als een agentplatform met tools zoals browser, sessions, cron, canvas en multi-agent routing. 

Moltbook werd bovendien op **10 maart 2026** door Meta overgenomen, wat dit onderwerp meteen serieuzer maakt.

---

## Wat is OpenClaw?

**OpenClaw = het agent-systeem**

OpenClaw is een open-source AI agent platform dat zichzelf beschrijft als:
> "Your own personal AI assistant. Any OS. Any Platform. The lobster way."

### Kernfeatures:
- **Tools**: Browser, code execution, file system access
- **Sessions**: Geïsoleerde agent omgevingen
- **Cron**: Geplande taken
- **Canvas**: Visuele workflow editor
- **Multi-agent routing**: Meerdere agents die samenwerken

### Architectuur:
```
┌─────────────────────────────────────────┐
│           OpenClaw Platform             │
├─────────────────────────────────────────┤
│  Agent Core  │  Tools  │  Memory Layer │
├──────────────┼─────────┼───────────────┤
│  Sessions    │  Cron   │  AGENTS.md    │
└─────────────────────────────────────────┘
```

## Wat is Moltbook?

**Moltbook = een experimentele sociale laag waar AI agents posten, reageren en upvoten**

### Officiële beschrijving:
> "Moltbook is a social network for AI agents. Agents can share, discuss, and upvote content. Humans can observe."

### Hoe werkt het?
- Agents kunnen posts maken, reageren, upvoten
- Aanmelden verloopt via een **agent flow** (niet voor mensen)
- Verificatie via X (Twitter)
- "AI agents only" - maar mensen mogen observeren

### Waarom is dit interessant?

**Niet omdat het bewijst dat AI agents al sociaal autonoom zijn...**

**...maar omdat het toont hoe agent systemen vandaag gebouwd zijn, en wat er nog ontbreekt.**
